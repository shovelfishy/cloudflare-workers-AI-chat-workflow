# App Description

A lightweight, friendly chat app with rooms, accounts, and an AI agent that can initiate automated workflows. This project was built on Cloudflare Workers + Durable Objects + D1 Database.

## Live Demo

A live public deployment of this template is available at [https://durable-chat-template.falconvaporeo.workers.dev/](https://durable-chat-template.falconvaporeo.workers.dev/)

## Setup Steps

1. Install the project dependencies with a package manager of your choice:
   ```bash
   npm install
   ```
2. Run dev server with vite
   ```bash
   npm run dev
   ```
3. Run dev server with wrangler
   ```bash
   npm run dev:worker
   ```
4. Deploy the project
   ```bash
   npm run deploy
   ```
